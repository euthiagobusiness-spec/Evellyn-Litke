# Página de Obrigado - Método Referência Cristã

Página estática criada para confirmar a inscrição e levar o lead para o grupo
oficial de WhatsApp.

## Como visualizar

Abra `obrigado.html` no navegador.

## Campos pendentes

Substitua estes placeholders quando tiver as informações finais:

- `[NOME DO EVENTO]`
- `[DATA]`
- `[HORÁRIO]`
- `[VÍDEO DA EVELLYN]`
- `[THUMBNAIL DO VÍDEO]`
- `[NOME DO GRUPO]`
- `[POLÍTICA DE PRIVACIDADE]`
- `[TERMOS DE USO]`
- `[CONTATO DE SUPORTE]`

## Rastreamento

O arquivo `obrigado.js` envia eventos para `window.dataLayer`, pronto para
Google Tag Manager ou integração equivalente.
