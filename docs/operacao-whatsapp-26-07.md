## Enquete de ativação

Pergunta: **Qual é hoje o maior obstáculo para você se tornar referência no digital?**

1. Minha mensagem ainda não é clara.
2. Produzo conteúdo, mas não sou reconhecido(a).
3. Tenho conhecimento, mas não gero oportunidades.
4. Tenho receio de crescer e comprometer meus valores.

Usar a resposta majoritária como tema dos Stories, do retargeting e da abertura da aula. Não exportar nomes ou números dos participantes para planilhas abertas.

## Proteção de dados e moderação

- Não copiar a lista de telefones para canais não autorizados.
- Não publicar prints com nomes ou números visíveis.
- Não usar o grupo para ofertas fora das finalidades aceitas no cadastro.
- Honrar pedidos de saída e retirada de consentimento.
- Restringir administradores ao mínimo necessário.
- Se o convite vazar, registrar o horário, substituir o link e atualizar a configuração central do site antes de retomar tráfego.
